## Linux终端

文件传递（Windows -> Linux）：Windows下直接拖动文件到VMware虚拟机界面

`ls`：查看当前路径下的文件

`pwd`：返回当前路径

`cd ..`：退到路径上一层

`鼠标左键双击选中+鼠标右键`：复制选中内容

`cd xxx+tab`：自动补充为当前路径下的文件名（在前缀唯一的情况下）

`tree` ：以树状结构显示目录内容

- 安装 `tree` 工具：`sudo yum install tree -y`

- `tree -L 2`：限制显示的层级深度为 2 层（当前目录文件和文件夹，及子目录内容）

## Gvim

`i`：==插入模式==，即允许修改文件

`ESC`：退出插入模式（假设叫==查看模式==）

- `u`：`ESC`模式下，**回退**到上一步操作（插入了错误内容）

- `:wq`：`ESC`模式下，`w`**保存/写入**，`q`**退出**

- `:%s/原内容/替换内容/g`：`ESC`模式下，文件**所有**原内容**替换**为替换内容
- `yxy`：`ESC`模式下，从光标所在位置，向下**复制**`x`行内容
- `p`：`ESC`模式下，**粘贴**复制内容（复制粘贴也可使用**Linux终端方式**）
- `dd`：`ESC`模式下，**剪切**当前光标所在的整行
- `gg`：`ESC`模式下，**回到文件首行**开头
  - `xgg`：跳转到文件第`x`行
- `G`：`ESC`模式下，**回到文件最后**一行开头
- `/abc`：`ESC`模式下，**查找**文件中`abc`文本
  - 配合`gg`与`G`确定搜索开始位置
  - `n`：**从上往下**查找
  - `N`：**从下往上**查找
